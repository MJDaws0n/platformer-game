# Platformer Game - Novus

A scrolling Linux platformer written mostly in Novus, using `novus-window` for the real native window and `novus-physics` for collision checks.

## Quick Start

```bash
cd ~/Desktop/platformer-game
nox init
novus main.nov
./build/linux_x86_64/main
```

## Project Structure

```
platformer-game/
├── main.nov              # Playable platformer entry point
├── libraries.conf        # generated by Nox
├── README.md
└── lib/                  # restored by nox init
```

## Libraries

### Reusable (External Repos)
- **physics** (novus-physics): AABB collision detection
  - GitHub: https://github.com/MJDaws0n/novus-physics
- **window** (novus-window): native Linux X11 window backend
  - GitHub: https://github.com/MJDaws0n/novus-window

## Features

- Scrolling Linux windowed platformer
- Left / right movement with jumping
- Platform collision via AABB checks
- Lives, respawn, goal flag, win/lose states

## Build

### Direct Compilation
```bash
novus main.nov --target=linux/amd64
./build/linux_x86_64/main
```

### With Nox (package manager)
```bash
nox init
novus main.nov
./build/linux_x86_64/main
```

## Controls

- **Left / A**: move left
- **Right / D**: move right
- **Up / W / Space**: jump
- **Esc or window close**: quit
- **Jump after win/lose**: restart

## Technical Notes

- `novus-window` builds a small Linux X11 helper on first launch
- The current Novus runtime still has a global-array startup bug, so level data is kept in scalar accessor functions for now
